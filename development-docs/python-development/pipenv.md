# :computer: Pipenv

:link: [Pipenv](https://pipenv.pypa.io/en/latest/)

## Install Pipenv

```shell
pip install --user pipenv
```

## Installing packages

```shell
cd project_folder
pipenv install packages_name
```

## Using installed packages

```shell
pipenv run python main.py
```

Start shell for project

```shell
pipenv shell
```
